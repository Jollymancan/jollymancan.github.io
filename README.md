# Personal Website

## Structure
- `index.html` — homepage
- `about.html`, `projects.html`, `blog.html`, `resume.html`
- `assets/` — CSS/JS/images
- `demos/` — interactive demos

## Local preview
Open `index.html` in a browser, or run a simple server:

Python:
python -m http.server 8000

Then visit http://localhost:8000
